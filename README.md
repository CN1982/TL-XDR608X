# 固件简要说明：
后台IP:192.168.1.1
官方OPENWET刷入后无线是关闭的，要自己手动开下，国家代码	也要选择下5G才正常。

固件每天0点自动编译。

固件信息里的时间为编译开始的时间，方便核对上游源码提交时间。
